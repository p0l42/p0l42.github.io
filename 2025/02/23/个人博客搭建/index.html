个人博客搭建

Hello，这里是s0l42，心血来潮搭建了一个博客，这篇用来记录基于github和hexo的博客搭建及过程中踩过的坑。

所需环境

环境：git、npm、node、hexo

软件安装过程自行百度~

依赖安装及配置

  1. 命令行中输入npm install hexo -g安装hexo
  2. 命令行中输入npm install hexo-deployer-git --save，用于后续向git更新文章

博客搭建

  1. github中新建一个repository

    坑点：仓库命名一定要为 用户名.github.io，否则会出现404的情况（TAT）

  2. 修改_config.yml文件中的deploy字段，可以参考下面的下面的内容

坑点：这里还有一个坑点，注意看branch是master还是main，需要和仓库的分支保持一致

3. 输入hexo g，生成html文件
tips: 为了防止推上去的文章有问题，可以先使用hexo s本地部署看一下效果

  1. 命令行输入hexo d将博客内容推送到github上，此时访问 用户名.github.io 就可以看到博客内容了~

后续计划

打算更博客美化以及如何写文章推送至博客内容，敬请期待~
